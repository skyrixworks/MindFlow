# Mindnote
Mindnote is a desktop note-taking application with AI-assisted features.

## Features (planned)
- Create and edit notes 
- Tagging, folder system, and linking between notes
- Graph visualization of note connections
- AI assistant for summarizing, structuring, and generating text
- Local or hybrid storage
- Automatic backups
- Planned external input via messaging apps or web APIs

## Status
This is an early-stage project. Features are being implemented incrementally.
